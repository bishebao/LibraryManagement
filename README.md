<h2>一、项目简介</h2>
基于ssm的图书管理系统：前端jsp、jquery，后端 servlet、jdbc，集成图书管理、图书分类管理、图书借阅、图书归还、公告、读者等功能于一体的系统。

<h2>三、功能介绍</h2>
<div class="markdown-heading" dir="auto">
<div class="markdown-heading" dir="auto">
<h3 dir="auto" tabindex="-1">管理员</h3>
<ul dir="auto">
 	<li>图书管理：图书信息的增删改查，图书名称关键词搜索</li>
 	<li>图书分类管理：图书分类信息的增删改查</li>
 	<li>图书借阅信息：读者借阅记录列表，还书操作</li>
 	<li>图书归还信息：归还信息列表，读者归还记录</li>
 	<li>公告管理：公告信息的增删改查</li>
 	<li>读者管理：读者信息的增删改查，密码设置用于读者登录系统时使用</li>
 	<li>我的：个人信息查看和修改，密码修改</li>
</ul>
<h3 dir="auto" tabindex="-1"><a id="user-content-读者" class="anchor" tabindex="-1" href="https://github.com/YuLin-Coder/No66LibraryManagementSystem2#%E8%AF%BB%E8%80%85" aria-hidden="true"></a>读者</h3>
<ul dir="auto">
 	<li>基本功能：登录、注册、退出、个人信息查看和修改，密码修改</li>
 	<li>图书查询：图书列表，关键词搜索，图书借阅操作</li>
 	<li>借阅信息：借阅操作后会在借阅信息记录里，读者可以自己还书操作</li>
 	<li>借阅历史：已经归还的图书借阅时间和归还时间，在借阅历史里可以查询</li>
</ul>
